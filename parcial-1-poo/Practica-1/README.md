Objetivo de la práctica
De el objetivo de esta practica seria el practicar como hacer clases, incluir otra clase dentro de una otra que en este caso es el index, como organizar metodos y funciones
• Descripción de la clase creada
Como descripcion tendriamos tenemos la clase usuario que se encargar de pedir el nombre y regresar ese valor, lo mismo con el email, ponemos valores vacios que en el index se llenaran.
Y en la clase del index ponemos el valor del nombre y email para despues mandarlo a llamar con el include Usuario que incluye los metodos usados teniendo algo mas organizado.

en mi caso que ya puse el valor dentro del index lo unico que hice fue abrir xamp /localhost el nombre de esta carpeta y el index se abre instataneamente mostrando el nombre e emaill que ya coloque desde un inicio
