Database es la conexion a la base de datos como ya bien se sabe y por la estructura de las carpetas estamos usando la estructura MVC con PHP PDO y MYSQL, el modelo se encarga de mandar datos al controller y que del controler le mande datos a views o mas especificamente la carpeta admin para que pueda leer, borrar, actualizar, insertar datos en mysql y que los cambios realisados esten tanto en mysql como en la pantalla para el usuario en este caso admin.
tenemos varios reads en este caso, un read es para consultar todos los datos dentro de la tabla mysql que podemos acceder gracias a modelos, el otro readd es para leer un solo dato a la vez ya que es seleccionado
el frm es el formulario donde el usuario/admin pondra los datos y estos datos seran transportados a la base de datos y almacenados, el delete elimina un dato en especifico en este caso y el archivo admin.php en este caso nos sirve como index
